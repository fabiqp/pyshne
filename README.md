# Welcome to Python Shell of New Era
## About
pyshne is a free and open-source shell written in python
## To test pychne
`git clone https://github.com/fabiqp/pyshne.git`

`cd pyshne`

`sudo ./test-pyshne`

`sudo chmod +777 pyshne`
## Installation 
To install pyshne execute these commands

`git clone https://github.com/fabiqp/pyshne.git`

`cd pyshne`

`sudo ./install`
